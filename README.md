# israel-flights-realtime-dashboard
This repository displays a real-time dashboard of all flights at Ben Gurion Airport


## Stack:
- Kafka Connect
- Kafka
- Spark 3.5.4
- Elastic search 8.1.7
- Kibana 8.1.7
- Java 17
- Spring boot 3.4.2
